A few projects that show how I approach UX, UI and front‑end engineering. All built with performance and maintainability in mind.

VAWC Alert Mobile App

VAWC Alert is a mobile-based incident reporting application designed to help individuals report cases of violence against women and children quickly and securely. The app allows users to send reports with photos, videos, and real-time location to ensure faster response during emergencies. It also includes a self-assessment feature and informational resources to help users recognize VAWC cases and seek proper support.

Dart Flutter Geolocator (Flutter) Flutter Map (OpenStreetMap), LatLong2 Firebase Cloud Firestore Firebase Authentication

VAWC Alert Admin Web

The VAWC Alert Admin Panel is a web-based system built with React and Vue, integrated with Firebase, designed to manage and monitor alerts sent by users in real time. It allows administrators to review incident reports, track alert statuses, and respond promptly to emergency cases. The system also handles account setup and approval, ensuring that only verified users can access and use the platform securely.

React Vue.js Firebase Cloud Firestore Firebase Authentication
